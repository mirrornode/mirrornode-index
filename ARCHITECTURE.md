# Architecture — mirrornode-index

## Role
mirrornode-index is the cross-repo reference index for MIRRORNODE. It provides a navigable map of repos, roles, statuses, and documentation references.

## What Lives Here
- Repo inventory
- Cross-repo dependency notes
- Links to canonical docs
- Contributor orientation references

## What Does Not Live Here
- Application code
- Agent runtime implementations
- Secrets or environment configuration
- Deployment-specific state

## Canonical Relationship
MIRRORNODE-CORE-HUB is authoritative for governance. This repo indexes and references that canon.
